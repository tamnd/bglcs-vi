# Hướng dẫn học Khoa học Máy tính của Beej (bản tiếng Việt)

Bản dịch tiếng Việt của [Beej's Guide to Learning Computer Science](https://beej.us/guide/bglcs/).

Đọc trực tuyến: **https://tamnd.github.io/bglcs-vi/**

## Về bản dịch

Nội dung tiếng Việt nằm trong thư mục `src_vi/`. Bản gốc tiếng Anh thuộc bản quyền
của Brian "Beej Jorgensen" Hall, phát hành theo giấy phép CC BY-NC-ND 4.0.
Bản dịch cũng theo giấy phép đó.

Dịch bởi Duc-Tam Nguyen.

## Build

```bash
git submodule update --init
bash scripts/build_vi.sh
```

Output will be in `docs/`.
